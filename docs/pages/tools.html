Tools¶

To facilitate the development work, several tools are provided, including:

  • rtmonitor A profiling tool built to monitor and capture real time performance metrics of ROS2 C++ Application
  • rdk_perf Performance tool for ROS2 topics and show result in Rqt at run time

For details, please refer to Tools Project.
